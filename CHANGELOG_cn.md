# 更新日志 (Changelog)

所有关于 Get笔记导入器 (GetImporter) 插件的显著变更都将记录在此文件中。

该格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
且本项目遵循 [语义化版本规范 (Semantic Versioning)](https://semver.org/spec/v2.0.0.html)。

## [3.0.0] - 2026-04-12

### 新增 (Added)
- 📄 **YAML 前方数据 (Properties)**: 将笔记元数据（标题、创建时间、标签）统一收纳进符合 Obsidian 官方标准的 YAML 属性区块中，不再是零散的正文文本。
- 📎 **通用附件兼容**: 针对原导出文件中的普通文件（如被 `<a>` 标签包裹的 PDF 等）新增完整的 Turndown 识别规则，使其能被无缝转存为 Obsidian 标准内部链接。

### 更改 (Changed)
- 📝 **标题表现形式 (ATX)**: 更新了 Markdown 渲染规则中的 `Heading` 选项，抛弃了容易串词的 `setext` (`======`) 风格，改为 Obsidian 最佳实践的标准井号 `atx` (`#`, `##`) 风格。

### 修复 (Fixed)
- 🐛 **移动端导入崩溃 (`undefined path`)**: 增加了通过原生 `FileReader` ArrayBuffer 向后兼容文件的提取逻辑，修复了在移动设备上因操作系统屏蔽底层路径导致选择 `.zip` 导入直接报错中止的问题。
- 🐛 **夹杂的有序/无序列表不换行**: 定位并修复了极高频出现的 Get笔记 嵌套列表（如 `<p>` 中内夹着 `* `、`- `、`1. ` 等）不被视作正确列表处理的排版问题。基于强化正则将其强制转换为标准列表。
- 🐛 **非法路径字符崩溃 (`ENOENT`)**: 修复了由于自动生成的笔记标题中含有被系统禁止的路径符号（如 `/`, `\`, `:`, `?` 等等）从而导致本地物理文件创建失败的黑洞 Bug，加入了安全字符清洗阶段。
- 🐛 **内嵌录音媒体丢失**: 修复了之前一处将 `div.attachment` 提前删去而跳过音频处理节点的代码，使诸如 `.mp3` 录音也能完美还原回原文中的 `![[ ]]` 内嵌播放器格式。
- 🐛 **图片附随狗皮膏药文件名**: 在提取 HTML 转 Markdown 的阶段前，通过代码摘除并移除了多余的 `.image-title` 这个 `span` 标签，还清纯粹的图文体验。
- 🐛 **Obsidian 空命名幽灵文件夹**: 摘除了配置路径最后的斜杠尾巴 `/get attachment/` -> `/get attachment` ，避开了导致 Obsidian API 在其下方误生成空子目录名的行为漏洞。
- 🐛 **多余不必要的反斜杠 (\)**: 在末期对 Turndown 的过度安全机制生成的各种反斜杠转义符号（如针对 `_` 星号 `*` 点号 `.` 形成的形如 `gk\_live`）进行一次全局替换和无害化清理，恢复绝佳的文本阅读体验。

## [2.0.0] - 2026-01-12

### 🎉 重大发布 - 全面品牌升级

这是一个大版本更新，插件从 "flomo" 全面品牌化更名为 "Get笔记"，并在用户界面上做出了重大改进。

### 新增 (Added)
- ✨ **最新 UI 设计**: 具有现代化且有序界面的全面插件重构设计
- ✨ **中文 UI 支持**: 插件面板界面中文化全覆盖
- ✨ **新图标**: 采用全新记事本风格 SVG 应用图标
- ✨ **同步状态视窗**: 实时显示您的同步状态、最后同步时间和相关笔记计数
- ✨ **清理同步历史开关**: 增加一键按钮清除一切同步残留数据并允许全局重置覆盖式导入
- ✨ **综合性全量文档**: 完全重写 README 开源主页介绍，并引入图文交互指南
- ✨ **开发者参与守则**: 为愿意打磨贡献代码的用户撰写了 CONTRIBUTING.md
- ✨ **未来产品路线图**: 增加 ROADMAP.md 展示中远期的功能研发预想表
- ✨ **底层架构文档**: 增加 ARCHITECTURE.md 诠释本项目设计与工程方法论
- ✨ **英文文档**: 提供面向全球用户的 README.en.md


### 更改 (Changed)
- 🔄 **全局重新命名**: 处理了包含 "flomo" 的一切命名空间为 "Get" 亦或 "Get笔记"：
  - 目录架构: `lib/flomo/` → `lib/get/`
  - 核心类: `FlomoImporter` → `GetImporter`, `FlomoCore` → `GetCore` 等等
  - 变量设置: `flomoTarget` → `getTarget`
- 🎨 **面板设置块优化**: 进行整理归类，模块更为清晰:
  - 手动导入 (Manual Import)
  - 基础设置 (Basic Settings)
  - 视觉效果选项 (Visualization Settings)
  - 高级选项 (Advanced Options)
  - 自动同步 (Auto Sync)
  - 数据管理 (Data Management)

### 修复 (Fixed)
- 🐛 **白板构建路径 Bug**: 修复了由于寻址异常导致画布最终为透明或空白的问题
- 🐛 **时间轴瞬间逆转**: 修正日期读取参数确保最新的信息流顶置在前
- 🐛 **HTML 树报错**: 通过兼容方案解决了被卡退出现的 `cloneNode is not a function` 依赖问题
- 🐛 **自动劫持登入**: 修正了登录的监控 URL 事件范围以保证静默提取
